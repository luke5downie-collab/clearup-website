# ClearUp Website

Marketing site for [ClearUp](https://clearup.com) — a daily acne tracker for iOS.

## Stack
Single static HTML page + two legal pages. No build step, no dependencies.

## Files
- `index.html` — landing page
- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Service

## Local preview
Open `index.html` in a browser. That's it.

## Deploy
Hosted on Vercel. Push to `main` → live.
